# Bilibili 字幕复制器

一个简单易用的浏览器脚本，用于在哔哩哔哩(Bilibili)视频页面左侧提供一键复制字幕按钮。脚本会优先复用 GreasyFork `378513` `Bilibili CC字幕工具` 的字幕下载弹窗；如果当前播放器版本无法稳定触发该弹窗，则自动回退到同一条 B 站字幕接口并复制 `TXT` 纯文本结果。

https://github.com/learnerLj/bilibili-subtitle

## 功能特点

- 在Bilibili视频页面左侧添加半透明粉色"复制字幕"按钮
- 优先复用 `378513` 脚本的字幕下载能力
- 当前播放器版本下会自动回退到同源字幕接口，避免被隐藏 DOM 卡住
- 直接复制字幕到剪贴板
- 默认复制 `TXT` 纯文本字幕，适合喂给大模型
- 简单易用，一键操作
- 下载过程中显示状态反馈
- 贴合B站界面风格的设计

## 安装方法

1. 首先安装用户脚本管理器（如 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)）
2. 可以先安装 GreasyFork `378513` [Bilibili CC字幕工具](https://greasyfork.org/zh-CN/scripts/378513-bilibili-cc%E5%AD%97%E5%B9%95%E5%B7%A5%E5%85%B7) 以复用它现有的字幕处理链路
3. 安装本仓库的脚本
4. 访问任意Bilibili视频页面，脚本将自动激活

## 使用方法

1. 打开任意带有字幕的Bilibili视频
2. 如果已经安装 `378513`，脚本会优先尝试调用它的字幕下载弹窗
3. 在视频页面左侧中央位置找到粉色"复制字幕"按钮
4. 点击按钮后，脚本会先尝试走 `378513` 的弹窗路径
5. 如果当前播放器版本无法稳定打开该弹窗，脚本会自动回退到同源字幕接口并直接复制 `TXT`
6. 成功时会在按钮旁边显示复制结果提示
7. 复制完成后，可以将字幕粘贴到任意文本编辑器中保存

## 界面特点

- 按钮位于视频页面左侧中央位置
- B站粉色半透明设计，不干扰视频观看
- 点击时显示复制中状态，提供清晰的操作反馈
- 成功提示显示在按钮旁边，不遮挡视频内容

## 注意事项

- 该脚本需要视频包含字幕才能正常工作
- 该脚本可与 GreasyFork `378513` `Bilibili CC字幕工具` 配合使用，但不再强制依赖它
- 如果当前播放器版本的字幕面板 DOM 与 `378513` 的旧触发路径不兼容，脚本会自动走回退逻辑
- 如果视频本身没有字幕轨，本脚本会直接报错
- 复制内容固定为 `TXT` 纯文本，不带时间轴

## 技术要求

- 支持用户脚本的现代浏览器（Chrome、Firefox、Edge等）
- 已安装Tampermonkey或类似的用户脚本管理器
